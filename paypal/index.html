Comment payer via PayPal

Introduction

Le prix pour une bannière est de 1,5€, vous pouvez payer via PayPal en suivant les étapes suivantes :

Étape 0

Vous devez avoir un compte PayPal valide, si vous n'en avez pas, vous pouvez en créer un en cliquant ici.

Étape 1

Cliquez sur le bouton en bas de cette page, vous serez redirigé vers la page de transfère de PayPal.

Étape 2

Sur cette page, cliquez sur le bouton "Envoyer"

Étape 3

Entrez la somme de 1,5€, et dans la case de la nature de payement, entrez :

"Demande de bannière style [le numéro du style que vous souhaitez]. Mon pseudo Discord est : [votre pseudo Discord]"

Puis cliquez sur "Continuer" et suivez les instructions données par PayPal.

ATTENTION !

Tout payement inférieur à 1,5€ ne sera pas accepté (sauf réduction obtenue par Truzz et uniquement par lui sur Discord)

Cependant tout montant au dessus de 1,5€ est le bienvenu !

Payez dès maintenant avec PayPal !
